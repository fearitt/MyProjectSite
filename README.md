Шаблон для сайта ресторана бургеров на платформе 1C-Битрикс.

## 🚀 Особенности
- Адаптивный дизайн
- Страницы: Главная, Меню, О нас, Блог, Контакты
- Слайдер товаров
- Форма обратной связи
- Социальные сети
